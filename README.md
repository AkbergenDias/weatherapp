## Features

- **Multi-city Navigation:** Smooth page-turning transitions using `UIPageViewController` with synchronized tab bar indicators.
- **City Search:** Location search management with dynamic data updates.
- **Adaptive UI:** Fully programmatic layout designed to handle multiple screen sizes seamlessly.

## Tech Stack & Architecture

- **Language:** Swift
- **Framework:** UIKit (100% Programmatic UI)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Layout Engine:** SnapKit
- **IOS version:** 15.6+
- **Concurrency:** Grand Central Dispatch (GCD)
